## Common Pitfalls
*   **Non-Stationary Data:** Ensure time series data is stationary before analysis. Use differencing or other transformations if needed.  The `adfuller` function helps check for stationarity.
*   **Objective Function Choice:** The objective function (e.g., MSE) significantly impacts calibration results. Experiment with different functions.
*   **Parameter Bounds:** Define appropriate bounds for parameters to ensure realistic values and prevent optimization errors.
*   **ABM Complexity:** Start with a simplified ABM and gradually increase complexity after successful calibration.
*   **Data Scaling:** Scale data appropriately to improve optimization performance.
*   **Local Minima:** Optimization algorithms can get stuck in local minima. Try different initial guesses or global optimization methods.
*   **Simulation Variance:** Account for stochasticity in the ABM by running multiple simulations for each parameter set and averaging the results.
