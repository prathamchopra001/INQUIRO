Implement a competitive pricing strategy in an Agent-Based Model (ABM) where firms observe and react to competitors' prices.

## DO NOT USE
*   `random.choice` for weighted random selection. Use `numpy.random.choice`.
*   Global variables without clear justification. Encapsulate state within agent or model classes.
*   Hardcoded parameters. Define parameters in the model initialization and pass them to agents.

## Example Code

```python
import mesa
import numpy as np

class Firm(mesa.Agent):
    def __init__(self, unique_id, model, initial_price=10):
        super().__init__(unique_id, model)
        self.price = initial_price

    def step(self):
        # Observe competitor prices (simplified: average price)
        competitor_prices = [agent.price for agent in self.model.schedule.agents if agent != self]
        if competitor_prices:
            avg_competitor_price = np.mean(competitor_prices)
            # React to competitor prices (example: adjust price towards the average)
            self.price += 0.1 * (avg_competitor_price - self.price)
        else:
            # Handle the case of no competitors (first firm)
            pass

class Model(mesa.Model):
    def __init__(self, N=2, width=10, height=10):
        self.num_agents = N
        self.grid = mesa.space.MultiGrid(width, height, True)
        self.schedule = mesa.time.RandomActivation(self)
        self.running = True

        # Create agents
        for i in range(self.num_agents):
            a = Firm(i, self)
            self.schedule.add(a)
            # Place the agent in a random location
            x = self.random.randrange(self.grid.width)
            y = self.random.randrange(self.grid.height)
            self.grid.place_agent(a, (x, y))

    def step(self):
        self.schedule.step()

# Example usage
model = Model(N=5)
for i in range(10):
    model.step()
    prices = [agent.price for agent in model.schedule.agents]
    print(f"Step {i+1}: Prices = {prices}")
```

## Common Pitfalls
*   **Incorrect Price Aggregation:** Ensure the `competitor_prices` list is properly filtered to exclude the agent's own price.  Using `if agent != self` is crucial.
*   **Division by Zero:** Handle cases where there are no competitors (e.g., the first firm in the simulation).
*   **Unrealistic Price Fluctuations:** Add damping factors (e.g., the `0.1` in the example) to prevent prices from oscillating wildly.
*   **Missing Agent Placement:** Ensure agents are placed on the grid using `self.grid.place_agent`.
*   **Infinite Loops:** Ensure that the price adjustment mechanism doesn't lead to an infinite loop or unrealistic price values (negative prices, extremely high prices). Consider clipping price changes.
