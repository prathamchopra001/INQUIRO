## Example Code
```python
# Assume abm_forecasts and actual_data are pandas Series
# and var_forecasts is a numpy array

def calculate_rmse(predictions, actual):
    return np.sqrt(mean_squared_error(actual, predictions))

def calculate_mae(predictions, actual):
    return mean_absolute_error(actual, predictions)

def diebold_mariano_test(actual, forecast1, forecast2, h=1):
    """
    Returns the Diebold-Mariano test statistic and p-value.
    """
    e1 = actual - forecast1
    e2 = actual - forecast2
    d = e1**2 - e2**2
    mean_d = np.mean(d)
    var_d = np.var(d)
    T = len(actual)
    stat = mean_d / np.sqrt((var_d / T))
    p_val = 2 * norm.cdf(-np.abs(stat))
    return stat, p_val


# Example Usage (assuming abm_forecasts, var_forecasts, actual_data exist):
abm_rmse = calculate_rmse(abm_forecasts, actual_data)
var_rmse = calculate_rmse(var_forecasts, actual_data)

abm_mae = calculate_mae(abm_forecasts, actual_data)
var_mae = calculate_mae(var_forecasts, actual_data)

dm_stat, dm_pval = diebold_mariano_test(actual_data, abm_forecasts, var_forecasts)

print(f"ABM RMSE: {abm_rmse}, VAR RMSE: {var_rmse}")
print(f"ABM MAE: {abm_mae}, VAR MAE: {var_mae}")
print(f"Diebold-Mariano Statistic: {dm_stat}, p-value: {dm_pval}")


# VAR example
data = pd.DataFrame({'y': actual_data, 'x': abm_forecasts}) #Example data
model = VAR(data)
results = model.fit(maxlags=5, ic='aic')  # Choose lag order using AIC
forecast = results.forecast(data.values[-results.k_ar:], steps=1) # One-step ahead forecast
var_forecasts = forecast[:,0] # Extract 'y' forecast
```

## Common Pitfalls
*   **Data Alignment:** Ensure `abm_forecasts`, `var_forecasts`, and `actual_data` are aligned by index/time.  Use `pandas.Series.reindex()` or `pandas.DataFrame.align()` if needed.
*   **Stationarity:** VAR models require stationary data. Apply differencing (`pandas.Series.diff()`) if necessary, and remember to reverse the differencing after forecasting.
*   **Forecast Horizon:** The `h` parameter in `diebold_mariano_test` represents the forecast horizon. Set correctly.
*   **VAR Lag Order:** Select the appropriate lag order for the VAR model using information criteria (AIC, BIC) and check residual diagnostics.
*   **Diebold-Mariano test assumptions**: The Diebold-Mariano test assumes that the forecast errors are stationary and serially uncorrelated. Verify the assumptions.
