## Common Pitfalls
*   **Incorrect Parameter Grid:** Ensure all parameter combinations are tested using `itertools.product`.
*   **Inefficient Simulation:** Optimize the ABM simulation code for speed, as sensitivity analysis involves running it many times.
*   **Ignoring Initial Conditions:** Be aware that initial conditions can affect results; consider running multiple simulations per parameter set.
*   **Lack of Visualization:** Failing to visualize the results can lead to misinterpretations.  Use `seaborn` or `matplotlib` to explore relationships.
*   **Overfitting to a Single Metric:** Evaluate the impact on convergence, pricing, and macroeconomic performance.
*   **Not Reporting Statistical Significance:** Statistical tests (e.g., ANOVA) should be used to confirm if parameter differences are statistically significant.
