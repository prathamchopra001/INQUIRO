# Skill: Heterogeneity of Pricing Strategies

## Task Description
Measure the heterogeneity of learned pricing strategies across firms of different sizes (small, medium, large) by calculating the standard deviation of prices within each size category.

## Required Techniques
Data aggregation, Standard deviation calculation

## Output Format
A dictionary or pandas DataFrame showing the standard deviation of prices for each size category (small, medium, large).

## Instructions

### 1. Required Packages
```bash
pip install pandas numpy
```

### 2. Correct Imports
```python
import pandas as pd
import numpy as np # for nan handling when calculating std
```

### 3. DO NOT USE
*   Manual looping for aggregation: Use `groupby()` instead.
*   `ddof=0` unless specifically required: Default `ddof=1` provides an unbiased estimate of the population standard deviation when working with samples.
*   Ignoring `NaN` values: Handle missing data correctly.

### 4. Example Code
```python
import pandas as pd
import numpy as np

# Sample DataFrame (replace with your actual data)
data = {'firm_size': ['small', 'small', 'medium', 'medium', 'large', 'large', 'small', 'medium'],
        'price': [10, 12, 15, 16, 20, 22, 11, 17]}
df = pd.DataFrame(data)

# Calculate standard deviation of prices for each firm size
price_std = df.groupby('firm_size')['price'].std()

# Print the result
print(price_std)

# Alternatively, store it in a dictionary if needed
price_std_dict = price_std.to_dict()
print(price_std_dict)


# NaN handling example (if price data has missing values)
data_with_nan = {'firm_size': ['small', 'small', 'medium', 'medium', 'large', 'large'],
                'price': [10, 12, 15, np.nan, 20, 22]} #using numpy NaN
df_nan = pd.DataFrame(data_with_nan)

price_std_nan = df_nan.groupby('firm_size')['price'].std()
print(price_std_nan)
```

### 5. Common Pitfalls
*   **Incorrect `groupby()` usage:** Ensure you are grouping by the correct column ('firm_size' in this case).
*   **Missing Data:** Use `numpy.nan` for missing data and let pandas handle it during the standard deviation calculation (it automatically excludes NaNs). If necessary, impute missing values *before* calculating the standard deviation.
*   **Empty Groups:** Handle cases where a firm size category might have no price data.  The `.std()` method will return `NaN` in such cases, so consider handling them (e.g., filling with 0 or another appropriate value).
*   **Data Type:** Ensure the 'price' column is numeric. If it's not, convert it using `df['price'] = pd.to_numeric(df['price'])`.
*   **Zero Standard Deviation:** If a group has only one unique price, the standard deviation will be zero.  This is mathematically correct, but consider if this scenario requires special handling in your analysis.
