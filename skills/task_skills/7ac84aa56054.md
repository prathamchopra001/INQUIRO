## Example Code

```python
import pandas as pd
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler
import numpy as np

# Sample data (replace with your actual data)
data = {'firm_id': range(100),
        'avg_price': np.random.rand(100) * 50,  # Example pricing data
        'price_std': np.random.rand(100) * 10,   # Example pricing data
        'size': np.random.randint(10, 100, 100), # Example firm size
        'sector': np.random.choice(['A', 'B', 'C'], 100)} # Example sector

df = pd.DataFrame(data)

# 1. Prepare data for clustering (pricing features)
pricing_features = ['avg_price', 'price_std']
X = df[pricing_features]

# 2. Scale the data
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 3. Determine optimal number of clusters (Elbow method - example)
# In real application, implement the elbow method to find optimal k
n_clusters = 3 #example, use the elbow method or other technique to find the optimal number.

# 4. Apply KMeans clustering
kmeans = KMeans(n_clusters=n_clusters, random_state=42, n_init='auto') #n_init avoids a warning
df['cluster'] = kmeans.fit_predict(X_scaled)

# 5. Analyze cluster characteristics
cluster_summary = df.groupby('cluster').agg(
    avg_price=('avg_price', 'mean'),
    size=('size', 'mean'),
    sector=('sector', lambda x: x.value_counts().index[0])  # Most frequent sector
)

print(cluster_summary)
```

## Common Pitfalls
*   **Incorrect Data Input:** Ensure the `avg_price` and `price_std` columns exist and contain numerical data.
*   **Missing Scaling:** Failing to scale the data (`StandardScaler`) can lead to biased clustering results.
*   **Suboptimal `k`:** Choosing the wrong number of clusters (`n_clusters`) can misrepresent the underlying patterns. Use techniques like the Elbow method or silhouette analysis to guide the choice of `k`.
*   **`n_init` warning:** KMeans requires specifying the `n_init` parameter to avoid a warning. Setting it to 'auto' is a good default.
*   **Random State:** Always set a `random_state` for reproducibility.
*   **Dataframe Modification:** Double-check that the cluster assignments are correctly added to the original dataframe.
*   **Feature Selection:** Carefully select the features used for clustering. Irrelevant features can negatively impact the results.
