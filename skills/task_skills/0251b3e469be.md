## Example Code

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import random

class Firm:
    def __init__(self, price=10, cost_change=0.1): # cost_change is the cost of changing price
        self.price = price
        self.cost_change = cost_change

    def adjust_price(self, new_price):
        # Decide whether to adjust price based on the cost
        if self.cost_change == 0:
            self.price = new_price
            return True

        if abs(new_price - self.price) > self.cost_change:
            self.price = new_price
            return True # Price adjusted
        else:
            return False # Price not adjusted

# Example Simulation
num_firms = 100
firms = [Firm() for _ in range(num_firms)]
prices = []

for t in range(100):
    # Simulate demand shock
    demand_shock = np.random.normal(0, 1)
    adjusted_count = 0

    for firm in firms:
        # Firms react to demand shock (simplified)
        new_price = firm.price + demand_shock * 0.5
        if firm.adjust_price(new_price):
            adjusted_count +=1

    prices.append(np.mean([f.price for f in firms]))
    print(f"Time {t}: Avg Price = {prices[-1]}, Adjusted Firms = {adjusted_count}")


# Basic Time Series Plot
plt.plot(prices)
plt.xlabel("Time")
plt.ylabel("Average Price")
plt.title("Average Price over Time")
plt.show()
```

## Common Pitfalls

*   **Incorrect Cost Implementation:** Ensure the cost of changing prices is correctly factored into the firm's decision.
*   **Ignoring Heterogeneity:** Firms may have different cost structures or react differently to shocks.
*   **Oversimplified Price Adjustment:**  The price adjustment mechanism should realistically reflect firm behavior.
*   **Lack of Sensitivity Analysis:**  Vary the `cost_change` parameter to observe its effect on price adjustments and overall volatility.
*   **Insufficient Simulation Length:**  Run the simulation for a sufficiently long period to observe meaningful results.
*   **Not Tracking Adjusted Counts:** The code needs to track the *number* of firms that changed their prices in a period.
