## Example Code
```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from scipy.optimize import minimize

# Define the macroeconomic model (simplified example)
def macroeconomic_model(params, learning_firm_proportion):
    """
    A simplified macroeconomic model.
    Params: A list of parameters [param1, param2, ...]
    learning_firm_proportion: Proportion of learning firms in the economy.
    Returns: A scalar representing a macroeconomic forecast (e.g., GDP growth).
    """
    param1, param2 = params
    gdp_growth = param1 * learning_firm_proportion - param2 * (1 - learning_firm_proportion)
    return gdp_growth

# Define the objective function (e.g., maximizing GDP growth)
def objective_function(learning_firm_proportion, params):
    return -macroeconomic_model(params, learning_firm_proportion) # Negative for maximization

# Parameter sweep
params_range = {'param1': [1, 2, 3], 'param2': [0.5, 1, 1.5]} # Example ranges

results = []
for param1 in params_range['param1']:
    for param2 in params_range['param2']:
        params = [param1, param2]
        # Optimization to find optimal learning_firm_proportion
        result = minimize(objective_function, 0.5, args=(params,), bounds=[(0, 1)]) # Starting guess of 0.5
        optimal_proportion = result.x[0]
        max_gdp_growth = -result.fun
        results.append({'param1': param1, 'param2': param2, 'optimal_proportion': optimal_proportion, 'max_gdp_growth': max_gdp_growth})

results_df = pd.DataFrame(results)
print(results_df)

# Basic Visualization
plt.figure(figsize=(10, 6))
plt.scatter(results_df['optimal_proportion'], results_df['max_gdp_growth'])
plt.xlabel('Optimal Proportion of Learning Firms')
plt.ylabel('Maximum GDP Growth')
plt.title('Sensitivity Analysis Results')
plt.grid(True)
plt.show()
```

## Common Pitfalls
*   **Incorrect Objective Function:** Ensure the objective function is correctly defined for optimization (minimization or maximization).
*   **Missing Bounds:**  Optimization can fail without appropriate bounds on parameters (e.g., `learning_firm_proportion` between 0 and 1).
*   **Ignoring Convergence Warnings:**  Pay attention to convergence warnings from the optimization routine; adjust parameters or the optimization method if necessary.
*   **Insufficient Parameter Range:** Choose parameter ranges that adequately cover the plausible space of values.
*   **Lack of Documentation:**  Clearly document the model, parameters, and assumptions.
*   **Incorrect data types:** Ensure that the parameters passed to the model have the correct data types.
*   **Forgetting to negate the function**: The `scipy.optimize.minimize` function *minimizes* a function.  If you want to maximize, return the negative of the function.
