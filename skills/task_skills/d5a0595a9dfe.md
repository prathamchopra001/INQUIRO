## Common Pitfalls
*   **Incorrect Reward Function:** Ensure the reward signal accurately reflects relative performance.  A poorly designed reward function will lead to poor learning.
*   **Non-Stationary Environment:** The ABM is dynamic. Implement mechanisms (e.g., decaying learning rate) to handle the changing environment.
*   **Slow Convergence:** Tune hyperparameters (learning rate, discount factor, exploration rate) to optimize convergence speed. Consider using eligibility traces for SARSA.
*   **Action Space Design:**  The action space (possible price changes) must be carefully designed to allow for meaningful learning.
*   **State Representation:**  While this example uses the firm ID as a state, consider more informative state representations (e.g., market share, inventory levels).
