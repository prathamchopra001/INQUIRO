## Common Pitfalls
*   **Data Stationarity:** Ensure time series data is stationary before fitting VAR models. Use differencing or other transformations if necessary. Test using `sm.tsa.stattools.adfuller`.
*   **Lag Order Selection:**  Choosing an appropriate lag order for the VAR model is crucial. Use information criteria like AIC, BIC, or FPE. `model.select_order()` helps.
*   **ABM Calibration:**  The ABM needs to be properly calibrated to match historical data.  This often involves parameter estimation techniques.
*   **Mesa Integration:**  Understand how to use Mesa's scheduler to run the ABM simulation and collect data at each step.
