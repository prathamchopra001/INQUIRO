```python
import mesa
import numpy as np

class Firm(mesa.Agent):
    def __init__(self, unique_id, model, size):
        super().__init__(unique_id, model)
        self.size = size
        self.price = 10  # Initial price

    def step(self):
        # Example pricing strategy update
        self.price += np.random.normal(0, 1)

class MarketModel(mesa.Model):
    def __init__(self, num_firms, size_influence):
        super().__init__(seed=42)
        self.num_firms = num_firms
        self.size_influence = size_influence # 0: none, >0: positive, <0: negative
        self.schedule = mesa.space.MultiGrid(width=10, height=10, torus=True)
        self.grid = mesa.time.RandomActivation(self)

        for i in range(self.num_firms):
            size = np.random.randint(1, 10) # Firm size
            firm = Firm(i, self, size)
            x = self.random.randrange(self.grid.width)
            y = self.random.randrange(self.grid.height)
            self.grid.place_agent(firm, (x, y))
            self.schedule.place_agent(firm, (x, y))
            self.schedule.add(firm)


    def step(self):
        self.schedule.step()

# Example usage:
model = MarketModel(num_firms=10, size_influence=0.5) # positive size influence
for i in range(10):
    model.step()
```

## Common Pitfalls
*   **Incorrect Matching Probability:** Ensure the `size_influence` parameter correctly modifies the probability of firms interacting, not just their behavior after an interaction. Consider using `numpy.exp` or a sigmoid function to map size to a probability influence.
*   **Data Collection Issues:** Use `DataCollector` to efficiently gather pricing strategies. Collect agent-level data within the `step()` function.
*   **Simulation Runtime:** ABMs can be slow. Optimize code and consider parallelization if necessary.
*   **Heterogeneity Measurement:** Use statistical measures like variance or Gini coefficient to quantify pricing strategy heterogeneity across firm sizes. Store firm sizes and prices at each time step for analysis.
*   **Missing Size Influence:** Implement the `size_influence` parameter in the `MarketModel` class and pass it to the agent and incorporate it into the matching probability calculation.
