## Task Description
Analyze the heterogeneity of learned pricing strategies across firms within an Agent-Based Model (ABM) by calculating the standard deviation of pricing decisions for firms of different sizes and in different sectors under different market structures.

## DO NOT USE
*   Iterating directly over DataFrame rows using `iterrows()` - it's slow. Use vectorized operations instead.
*   Hardcoding sector or size categories. Use variables or data-driven approaches.
*   Ignoring potential `NaN` values when calculating standard deviation.

## Example Code

```python
import pandas as pd
import numpy as np

# Sample Data (replace with your actual ABM data)
data = {'firm_id': [1, 2, 3, 4, 5, 6],
        'sector': ['A', 'A', 'B', 'B', 'A', 'B'],
        'size': ['Small', 'Large', 'Small', 'Large', 'Small', 'Large'],
        'market_structure': ['Competitive', 'Oligopoly', 'Competitive', 'Oligopoly', 'Competitive', 'Oligopoly'],
        'price': [10, 12, 15, 20, 11, 18]}
df = pd.DataFrame(data)

# Calculate standard deviation of pricing for each group
def analyze_pricing_heterogeneity(df, size=None, sector=None, market_structure=None):
    """
    Calculates the standard deviation of pricing decisions based on firm size, sector and market structure.
    """
    filtered_df = df.copy()
    if size:
        filtered_df = filtered_df[filtered_df['size'] == size]
    if sector:
        filtered_df = filtered_df[filtered_df['sector'] == sector]
    if market_structure:
        filtered_df = filtered_df[filtered_df['market_structure'] == market_structure]
    
    if len(filtered_df) > 1: #Std dev needs more than one value
        return filtered_df['price'].std()
    else:
        return np.nan # Or some default value
# Example usage
std_dev = analyze_pricing_heterogeneity(df, size='Large', sector='A', market_structure='Oligopoly')
print(f"Standard deviation of pricing for Large firms in sector A under Oligopoly: {std_dev}")

# Groupby approach (more flexible)
grouped_stats = df.groupby(['size', 'sector', 'market_structure'])['price'].std()
print("\nGrouped Standard Deviations:\n", grouped_stats)
```

## Common Pitfalls
*   **Zero Standard Deviation:** If a group has only one firm or all firms have the same price, the standard deviation will be zero. Handle this gracefully (e.g., return NaN or a custom message).  Addressed by checking group size.
*   **Missing Data:** Ensure your data doesn't have missing values that could skew the standard deviation.  Use `dropna()` or imputation if necessary *before* calculating statistics.
*   **Incorrect Filtering:** Double-check your filtering logic to ensure you're selecting the correct subsets of data.  Test your filters thoroughly.
*   **Data Types:** Ensure the 'price' column is numeric. If not, convert it using `df['price'] = pd.to_numeric(df['price'])`.
*   **Insufficient Data:** Handle cases where filtered groups are empty.
*   **Division by Zero:** Standard deviation calculation involves division. Ensure you aren't dividing by zero (e.g., when the sample size is 1).
