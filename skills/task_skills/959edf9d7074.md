## 4. Example Code

```python
import gymnasium as gym
from stable_baselines3 import A2C
from stable_baselines3.common.callbacks import BaseCallback
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import statsmodels.api as sm
from statsmodels.tsa.stattools import adfuller

class ReturnTracker(BaseCallback):
    def __init__(self, verbose=0):
        super(ReturnTracker, self).__init__(verbose)
        self.returns = []

    def _on_step(self) -> bool:
        info = self.locals.get("info", [{}])[0]
        if info.get("terminal_observation"):
            self.returns.append(info["final_info"]["episode"]["r"])
        return True


env = gym.make('CartPole-v1')
env = DummyVecEnv([lambda: env])  # Required for Stable-Baselines3

# Train with adaptive learning rate (Adam by default)
model = A2C('MlpPolicy', env, verbose=0)
return_tracker = ReturnTracker()
model.learn(total_timesteps=10000, callback=return_tracker)

returns = return_tracker.returns
returns_series = pd.Series(returns)

# Time Series Analysis for Stability
# 1. Rolling Statistics
rolling_mean = returns_series.rolling(window=30).mean()
rolling_std = returns_series.rolling(window=30).std()

# 2. Augmented Dickey-Fuller Test
result = adfuller(returns_series.dropna())
print('ADF Statistic: %f' % result[0])
print('p-value: %f' % result[1])
print('Critical Values:')
for key, value in result[4].items():
    print('\t%s: %.3f' % (key, value))

# Plotting
plt.figure(figsize=(12, 6))
plt.plot(returns_series, label='Episode Returns')
plt.plot(rolling_mean, label='Rolling Mean (30 episodes)')
plt.plot(rolling_std, label='Rolling Std (30 episodes)')
plt.legend()
plt.title('Episode Returns and Rolling Statistics')
plt.show()
```

## 5. Common Pitfalls
*   **Non-Stationary Returns:** RL returns often exhibit non-stationarity.  Differencing might be needed before applying ADF test.  Interpret the ADF test carefully; a low p-value suggests stationarity, indicating stability.
*   **Insufficient Training:**  Ensure the model trains for sufficient steps for the adaptive learning rate to converge. Monitor learning curves.
*   **Environment Variations:** Use the same environment seed when comparing different learning rate strategies.
*   **Incorrect Return Calculation:** Verify that episode returns are calculated correctly based on environment termination.
*   **Callback Issues:**  Double check the callback implementation to ensure returns are correctly captured at the end of each episode.
