## 4. Example Code

```python
import pandas as pd
from sklearn.cluster import KMeans
import statsmodels.api as sm
import statsmodels.formula.api as smf

# Load ABM simulation data (replace with your actual data loading)
data = pd.read_csv("abm_data.csv")  # Assuming CSV format, adjust accordingly

# Example: Clustering firms based on pricing strategy characteristics
X = data[['average_price', 'price_variance']] # Features for clustering
kmeans = KMeans(n_clusters=3, random_state=0, n_init=10) # Specify random_state and n_init
data['cluster'] = kmeans.fit_predict(X)

# Example: Regression analysis to determine the impact of firm size on average price
# Add a constant for the intercept
data['firm_size_sq'] = data['firm_size']**2
model = smf.ols("average_price ~ firm_size + firm_size_sq + market_share", data=data)
results = model.fit()
print(results.summary())

# Example: Statistical Analysis (e.g., comparing average prices across clusters)
print(data.groupby('cluster')['average_price'].mean())
```

## 5. Common Pitfalls

*   **Data Loading Issues:** Ensure the data loading mechanism (e.g., `pd.read_csv`) correctly parses the data file.  Verify column names and data types.
*   **Missing Data:** Handle missing values (NaN) appropriately before clustering or regression. Use `.fillna()` or drop rows with missing values `.dropna()`.
*   **Feature Scaling:** Consider scaling features (e.g., using `sklearn.preprocessing.StandardScaler`) before clustering if features have significantly different scales.
*   **Multicollinearity:** Check for multicollinearity in regression models using Variance Inflation Factor (VIF).  Address if present (e.g., remove highly correlated variables).
*   **Incorrect Formula Specification:** Double-check the formula used in `statsmodels.formula.api.ols` for syntax errors and correct variable names.  Use `Q` to quote column names with spaces or special characters, e.g., `smf.ols("average_price ~ Q('firm size')", data=data)`
*   **Interpretation:**  Carefully interpret regression coefficients and statistical significance.  Correlation does not equal causation.
*   **Random State:** Always specify a `random_state` for `KMeans` to ensure reproducibility. Also, specify `n_init` as the default is being deprecated.
