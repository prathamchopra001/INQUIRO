## DO NOT USE
*   `random.sample` for agent selection without considering agent attributes (use `np.random.choice` with probabilities).
*   Hardcoding parameters within agent classes. Pass them during initialization.
*   Ignoring the `step()` function in the agent and model classes.
*   Global variables for agent attributes.
*   Directly modifying agent lists during iteration (use `schedule.remove(agent)`).

## Example Code
```python
import mesa
import numpy as np
import pandas as pd
import statsmodels.api as sm
from statsmodels.tsa.arima.model import ARIMA

class Firm(mesa.Agent):
    def __init__(self, unique_id, model, size):
        super().__init__(unique_id, model)
        self.size = size
        self.price = 1.0

    def step(self):
        # Simulate price adjustment based on market conditions
        if self.model.schedule.time % 10 == 0: # Adjust price every 10 steps
          self.price += np.random.normal(0, 0.1)
          self.price = max(0.1, self.price) # Ensure price is not negative

class Market(mesa.Model):
    def __init__(self, num_firms, matching_probability_scale):
        super().__init__()
        self.num_firms = num_firms
        self.matching_probability_scale = matching_probability_scale # Larger firms more likely to match
        self.schedule = mesa.time.RandomActivation(self)

        # Create firms with varying sizes
        for i in range(self.num_firms):
            size = np.random.randint(1, 11) # Firm size between 1 and 10
            firm = Firm(i, self, size)
            self.schedule.add(firm)

        self.datacollector = mesa.DataCollector(
            model_reporters={"AveragePrice": lambda m: np.mean([a.price for a in m.schedule.agents])},
            agent_reporters={"Price": "price", "Size": "size"}
        )

    def step(self):
        self.datacollector.collect(self)
        self.schedule.step()

    def get_matching_probability(self, firm):
        # Example: Probability scales linearly with firm size
        return firm.size * self.matching_probability_scale

# Simulation Run
model = Market(num_firms=10, matching_probability_scale=0.1)
for i in range(100):
    model.step()

# Data Analysis
results = model.datacollector.get_model_vars_dataframe()
print(results.head())

# Time Series Analysis example
# Assuming you have enough data points
if len(results) > 10:
  try:
    arima_model = ARIMA(results['AveragePrice'], order=(5,1,0)) # Example ARIMA model
    arima_fit = arima_model.fit()
    print(arima_fit.summary())
  except Exception as e:
    print(f"ARIMA model failed: {e}")

agent_results = model.datacollector.get_agent_vars_dataframe()
print(agent_results.head())
```

## Common Pitfalls
*   **Incorrect Data Collection:** Ensure `DataCollector` reporters correctly extract relevant data. Verify the collected data types.
*   **Misconfigured ARIMA Models**: Ensure time series data is stationary before fitting ARIMA. Use appropriate `p, d, q` parameters.  Handle ARIMA exceptions gracefully.
*   **Inefficient Agent Matching:** Optimize agent matching algorithms for large-scale simulations.
*   **Ignoring Model Burn-in:** Discard initial simulation steps to avoid bias.
*   **Lack of Validation:** Validate the model against empirical data or theoretical predictions.
*   **Scale of `matching_probability_scale`:** Ensure the scale is appropriate. If too high, all firms will always match, and if too low, no firms will match. This makes the parameter ineffective.
