## Common Pitfalls
*   **Incorrect Parameter Ranges:** Choosing parameter ranges that are not meaningful for the problem.
*   **Insufficient Episodes:** Not running the simulation for enough episodes to observe the effect of parameter changes.
*   **Ignoring Variance:** Only focusing on the mean reward and neglecting the variance in the results. Consider using multiple runs per parameter set.
*   **Not Saving Results:** Failing to save the results of the sensitivity analysis for later analysis and comparison.
*   **Using complex environment with simple agent or vice versa**: Ensure the complexity of the agent and environment are balanced.
